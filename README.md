# remote light controller with web interface


### Installation

```bash
git clone https://github.com/polarspetroll/webGPIO.git; cd webGPIO;
chmod +x setup.sh
sudo ./setup.sh GPIO_PIN_NUMBER
go build main.go
./main
```

#### over WAN :

[portmap](https://portmap.io/)
[ngrok](https://ngrok.com/)

##### tested on :

- **Raspberrypi 4B+**
